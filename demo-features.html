How to Use Feature Flags

Add this script to your HTML:

<script src="js/featureFlags.js"></script>

Then use feature flags in your code:

// Check if feature is enabled if (featureFlags.isEnabled('feature_a')) { // Show feature A showUserDashboard(); } // Or use data attributes in HTML <div data-feature="feature_a"> This will only show if feature_a is enabled </div>

Environment Detection:

  • Stage URL: /stage/ → loads features.stage.json
  • Production URL: /production/ → loads features.production.json
